Lista desordenada

O elemento HTML <"ul"> (ou elemento HTML da lista desordenada) representa uma lista de itens sem ordem rigida, isto é uma coleção de itens que não trazem uma ordenação numérica e as suas posições, nessa lista, são irrelevantes.
